## [1.0.0] - 2025-12-11

### Added
- Initial release of Capy UI theme
- Complete dark theme with electric blue (`#1e80c7`) accents
- Comprehensive UI color definitions for all VS Code elements
- Syntax highlighting optimized for blue color palette
- Full terminal theming with ANSI color support
- Git decoration colors
- Debug toolbar theming
- Peek view and diff editor styling
- Markdown-specific syntax highlighting
- Language-specific enhancements (Python, JavaScript, TypeScript, React, CSS, JSON, SQL)

### Design Features
- Dark charcoal background (`#1c1d21`) for reduced eye strain
- Electric blue dominant accent color throughout
- Green strings (`#00ba7c`) for easy string identification
- Grey-blue function highlighting (`#93abd0`)
- Light grey-blue class/type names (`#a8b9ca`) for clear structure
- Bright teal symbols (`#5FF4AEFF`) for punctuation and operators
- Muted blue comments (`#8bb4d8`) with italic styling
- High contrast error red (`#e74c3c`)
- Accessible contrast ratios for all text elements
